## 全端測試使用
### 系統安裝
```cmd
npm install
npm install supervisor -g
```
### 執行測試
```cmd
npm start
```
---
## 測試產品發布
### 系統安裝
```cmd
npm install
```
## 執行服務
```cmd
npm run production
```
---
## 正式產品發布
### 系統安裝
```cmd
npm install
```
### 執行服務
```bat
SET NODE_ENV=production
node app.js
```
